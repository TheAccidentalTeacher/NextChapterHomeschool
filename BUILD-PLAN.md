## CANONICAL ECONOMY GLOSSARY

All phases, event effects, and implementation code **must** reference only these terms:

| Symbol | Name | Type | Notes |
|--------|------|------|-------|
| ⚙️ | Production | Economy resource | Generated by BUILD round |
| 🧭 | Reach | Economy resource | Generated by EXPAND round |
| 📜 | Legacy | Economy resource | Generated by DEFINE round; funds tech research |
| 🛡️ | Resilience | Economy resource | Generated by DEFEND round |
| 🌾 | Food | Survival dimension | Passive from Farm/Granary; not routed |
| 👥 | Population | Survival dimension | Grows/shrinks; yield multiplier + capacity gate |
| CI | Cultural Influence | Victory metric | Separate from resources; drives Cultural Victory |

**Prohibited legacy terms:** `gold`, `science`, `medicine` (as a resource — use "Medicine supply" for the purchasable item only), `movement` (use "Scout reveal range" or "EXPAND reach").
