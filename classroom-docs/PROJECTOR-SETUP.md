# Projector Setup — ClassCiv Realms

## Before every class

1. Open the projector tab at `/projector/[game-id]` on the second screen.
2. **Click the "Arm Ceremony" button once.** This unlocks the browser's
   audio context so the drumbeat and event sounds will play.
3. If you forget to click it: the game still works, but draft-reveal
   drumbeats and betrayal stings will be silent. A visual drum pulse
   (🥁) takes their place.

## If audio stops mid-class

- The browser can reclaim the audio context when the tab is backgrounded
  or the computer sleeps.
- Fix: click any non-navigation button on the projector tab once.
  The Arm Ceremony gesture does not need to be re-run; any click is enough.

## Projector display mode

- Full-screen the browser tab (F11 on Chrome/Edge, Cmd+Ctrl+F on Safari).
- Disable notifications on the projector computer before class.
- Keep the DM control computer on a separate tab/window.
